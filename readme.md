## ⚙️ How It Works

- Each batch/section has its own tracker file with a live progress table (GitHub, Repo, LeetCode links + live stats card per student)
- A **🏆 Weekly Leaderboard** section automatically updates every day via GitHub Actions — showing the top 3 solvers and students who haven't solved anything that week
- Data resets weekly (Monday–Sunday) so it's always a fresh race
